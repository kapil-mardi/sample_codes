Smooth Bezier Curve


This is a code for smooth bezier curve. It has 2 control points. The line drawn always pass through start point and end point.

cpx1 and cpy1 are used to control the curve at start point.

cpx2 and cpy2 are used to control the curve at end point.

Code : 

            var c = document.getElementById("myCanvas");
            var ctx = c.getContext("2d");
            let data = [[0,0],[10,10],[20,50],[30,40],[40,10],[50,100],[60,20],[70,30]]
            ctx.scale(2,2)
            ctx.beginPath();

            for(let i = 1; i < data.length; i++){
                let cpx1 = (data[i][0] + data[i - 1][0]) / 2
                let cpy1 = data[i - 1][1]
                let cpx2 = (data[i][0] + data[i - 1][0]) / 2
                let cpy2 = data[i][1]
                ctx.strokeStyle = "black";
                ctx.bezierCurveTo(cpx1, cpy1, cpx2, cpy2,data[i][0],data[i][1]);
                ctx.rect(cpx1,cpy1,2,2)
                ctx.rect(cpx2,cpy2,2,2)
                ctx.rect(data[i][0],data[i][1],2,2)
                ctx.stroke();
            }
